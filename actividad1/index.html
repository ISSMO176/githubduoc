Paso a Paso de Actividades

Paso 1: Verificar version de Git

Con el codigo "git -v" podemos ver la version actual de git con la que se esta trabajando.

Paso 2: Configurar Git

Con el codigo "git config --global -e" se abrira el editor de texto en donde se podran configurar el nombre, email y username.

Paso 3: Inicializar el Repositorio

Con el codigo "git init" se inicializa el repositorio en el cual se esta trabajando, es importante utilizar el codigo "git branch -m nonbre" para crear una nueva rama dentro del repositorio.

Paso 4: Probar el repositorio

Para efectos de prueba se crea por ejemplo un archivo Readme dentro de la carpeta, posteriormente se usa "git add .", luego con el codigo "git status" se verifica que el repositorio correctamente esta activo y el archivo Readme puede ser subido, en caso de que quiera subirse solo cierto tipo de archivos el "git add *.tipo de archivo".

Paso 5: Realizar Commit

Con el codigo "git commit -m "mensaje"" se suben los archivos en stage con el "add ." al repositorio de manera correcta junto con un mensaje, con "git commit --amend" se corrige el mensaje del ultimo commit realizado, en caso de que quiera cambiarse.

Paso 6: Restablecer archivos

En caso de que se borre o perdamos algun archivo, con "git checkout ." restablecemos los archivos que estaban subidos hasta el anterior repositorio de git.

Paso 7: Ver diferencias

Con el codigo "git diff" podemos ver las diferencias entre una anterior version del repositorio y la actual.

Paso 8: Configurar alias

Con el codigo "git config --global alias.l "log --online" configuramos el alias.
